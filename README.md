

## Usage

Wrap your widget with a Custom Animated Widget to obtain an animated button.

```dart
CustomAnimatedButton(
        
        child: Container(width: 200,
        height: 100,
        decoration: BoxDecoration(
          color: Colors.green,
          borderRadius: BorderRadius.circular(20)
        ),
        ),
      ),
```
